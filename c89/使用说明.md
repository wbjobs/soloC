# 太空无人机收集游戏 - 使用说明

## 📋 项目概述

这是一个3D太空无人机收集游戏，支持多人异步回合制。玩家控制无人机在太空中收集碎片，躲避障碍物，获取最高分数。

### 核心功能
- ✅ **3D太空场景** - 使用Three.js渲染的低轨道太空环境
- ✅ **多人异步回合制** - 支持4人同时游戏，每人每天15分钟操作时间
- ✅ **A*路径规划** - 无人机自动规划最优路径，躲避障碍物
- ✅ **热力图显示** - 显示碎片密集区域，帮助玩家决策
- ✅ **历史回放** - 回放游戏历史，分析策略
- ✅ **实时通信** - Go + WebSocket 实时状态同步
- ✅ **资源管理** - 无人机燃料和载重限制

## 🚀 快速开始

### 前置要求
1. Go 1.21+ (后端服务器)
2. 现代浏览器 (Chrome/Firefox/Edge)

### 安装步骤

#### 方法一: 一键启动 (Windows)
双击运行 `启动服务器.bat` 即可自动完成所有步骤！

#### 方法二: 手动启动

1. **安装Go** (如果未安装)
   ```bash
   winget install GoLang.Go
   ```

2. **下载依赖**
   ```bash
   go mod download
   ```

3. **启动服务器**
   ```bash
   cd backend
   go run main.go
   ```

4. **打开游戏**
   在浏览器中访问: `http://localhost:8080`

## 🎮 游戏玩法

### 基本操作
- **🖱️ 鼠标拖拽** - 旋转视角
- **🔄 滚轮** - 缩放视角
- **📍 点击地面** - 设置无人机目标位置

### 游戏机制
1. **无人机系统**
   - 🛢️ **燃料**: 飞行消耗燃料，燃料耗尽无法移动
   - 📦 **载重**: 收集碎片有重量限制，超重无法收集

2. **碎片系统**
   - 🟡 金色碎片 - 高价值，重
   - 价值越高，分数越多
   - 收集的碎片会计入载重

3. **障碍物**
   - 🔴 红色障碍物会阻挡无人机
   - A*算法会自动规划绕路路径

4. **时间槽系统**
   - ⏰ 每位玩家每天有15分钟操作时间
   - 时间每天重置

### 用户界面
| 区域 | 说明 |
|------|------|
| 左上HUD | 显示玩家名、分数、剩余时间 |
| 右上HUD | 显示燃料条、载重条 |
| 右侧 | 在线玩家列表 |
| 左下 | 操作说明 |
| 底部 | 功能按钮 |

### 功能按钮
- **显示热力图** - 显示碎片密度热力图
- **显示玩家** - 显示/隐藏其他玩家无人机
- **历史回放** - 打开回放面板观看历史

## 🔧 技术栈

### 后端
- **语言**: Go 1.21+
- **WebSocket**: gorilla/websocket
- **算法**: A*路径规划 (3D空间)

### 前端
- **3D渲染**: Three.js r128
- **UI**: 原生HTML/CSS/JavaScript
- **通信**: WebSocket

## 🎯 核心算法

### A*路径规划
- 支持3D空间寻路
- 自动检测和绕过障碍物
- 最优路径计算
- 网格式搜索空间

### 热力图计算
- 基于碎片密度计算
- 颜色梯度显示 (蓝->红)
- 实时更新

## 📝 开发说明

### 后端文件说明

1. **models.go**
   - Vector3: 3D向量类
   - Player: 玩家数据
   - Drone: 无人机状态
   - Debris: 碎片对象
   - Obstacle: 障碍物

2. **manager.go**
   - GameManager: 游戏主管理器
   - 游戏循环更新
   - 玩家加入/离开
   - 碎片收集检测
   - 历史快照保存

3. **pathfinding.go**
   - FindPath(): A*路径寻路
   - 优先队列实现
   - 碰撞检测

### 前端文件说明

1. **app.js**
   - SpaceGame: 游戏主类
   - Three.js场景初始化
   - WebSocket通信
   - 无人机、碎片渲染
   - 热力图显示
   - 历史回放系统

## 🚧 注意事项

1. **端口占用**: 服务使用8080端口，确保未被占用
2. **防火墙**: 确保防火墙允许WebSocket连接
3. **浏览器**: 建议使用最新版Chrome/Firefox获得最佳体验

## 🎮 游戏技巧

1. **看热力图** - 优先前往红色/黄色区域收集碎片
2. **注意燃料** - 不要飞太远，记得回基地补充
3. **控制载重** - 满载时返回基地卸载
4. **避开障碍** - 红色障碍物会增加飞行距离
5. **多人协作** - 和朋友分工收集不同区域

## 🐛 常见问题

**Q: 页面加载失败?**
- 检查服务器是否启动成功
- 确认端口8080未被占用
- 尝试刷新页面

**Q: 无人机不动?**
- 检查燃料是否耗尽
- 检查每日时间是否用完
- 确认点击的是可移动区域

**Q: 无法收集碎片?**
- 检查载重是否已满
- 确保无人机靠近碎片
