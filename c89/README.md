# 太空无人机收集游戏

低轨道太空区域3D回合制游戏，玩家部署无人机收集太空碎片。

## 功能特性

- 3D低轨道太空地图
- 4人异步回合制（每人每天15分钟操作时间）
- 无人机燃料、载重限制
- WebSocket实时通信
- 垃圾密度热力图
- A*路径规划避障
- 历史回放功能

## 技术栈

- 后端: Go + WebSocket
- 前端: Three.js + JavaScript
- 算法: A*路径规划

## 快速开始

### 后端启动

```bash
cd backend
go run main.go
```

### 前端访问

在浏览器中打开 `frontend/index.html`

## 游戏规则

1. 每个玩家每天有15分钟操作时间
2. 无人机有燃料限制，飞行消耗燃料
3. 无人机有载重限制，收集碎片有重量
4. 返回基地可补充燃料和卸载碎片
