# 网络攻防 - 双人在线回合制游戏

## 项目简介

这是一个基于 Node.js + Socket.IO + React 构建的双人在线回合制网络攻防游戏。玩家可以扮演攻击方或防御方，通过不同的策略进行对抗。

## 游戏玩法

### 攻击方式
- 🔍 **端口扫描** - 基础成功率高
- 💥 **缓冲区溢出** - 中等成功率
- 🎣 **钓鱼邮件** - 中等成功率

### 防御方式
- 🔥 **防火墙调整** - 有效克制端口扫描
- 🔧 **补丁更新** - 有效克制缓冲区溢出
- 📚 **员工培训** - 有效克制钓鱼邮件

### 游戏规则
- 每局共 5 回合
- 玩家轮流扮演攻击方和防御方
- 系统隐藏的弱点矩阵会影响攻击成功率
- 得分高者获胜

## 技术栈

**后端:**
- Node.js + Express
- Socket.IO (实时通信)
- JWT (身份认证)
- bcryptjs (密码加密)

**前端:**
- React 18
- Socket.IO Client

## 安装和运行

### 1. 安装依赖

```bash
# 安装后端依赖
npm install

# 安装前端依赖
cd client
npm install
```

### 2. 启动项目

#### 方式一：同时启动前后端（推荐）
```bash
# 在项目根目录运行
npm run dev
```

#### 方式二：分别启动
```bash
# 启动后端服务器 (端口 5000)
npm run server

# 另开一个终端，启动前端 (端口 3000)
npm run client
```

### 3. 访问游戏

打开浏览器访问: `http://localhost:3000`

## 功能特性

✅ **用户认证** - JWT 注册登录
✅ **匹配队列** - 自动匹配对手

## 开发说明

- 后端服务器运行在 `http://localhost:5000`
- 前端开发服务器运行在 `http://localhost:3000`
- Socket.IO 用于实时游戏状态同步
- 游戏数据存储在内存中（生产环境建议使用数据库）

## 注意事项

- 目前用户数据存储在内存中，服务器重启后会丢失
- JWT 密钥硬编码在代码中，生产环境应使用环境变量
- 建议使用两个不同的浏览器或隐身窗口来测试双人对战
