# 卫星轨道分析与可视化应用

基于 Dash + Plotly + Three.js 的卫星轨道分析与可视化应用，使用 SGP4 算法计算卫星轨道，并集成 NRLMSISE-00 大气密度模型进行大气阻力修正。

## 功能特性

1. **卫星数据输入**
   - 通过 NORAD ID 从 Celestrak 获取 TLE 数据
   - 支持上传本地 TLE 文件

2. **轨道计算与可视化**
   - 使用 SGP4 算法计算未来 7 天的卫星轨道
   - Mapbox GL 展示卫星地面轨迹（15 分钟步长）
   - Three.js 3D 地球仪展示卫星轨道面和实时位置

3. **大气阻力修正（精度改进）**
   - 集成 NRLMSISE-00 大气密度模型
   - 针对 < 300km 低轨道卫星特别优化，解决 > 50km/天误差问题
   - 实时计算大气阻力加速度并修正轨道位置
   - 大气密度随高度、太阳活动、地磁活动动态调整

4. **轨道精度分析**
   - 实时显示轨道高度、估计日误差和周误差
   - 展示大气阻力修正效果和修正量统计
   - 根据轨道高度提供精度改进建议

5. **地面站可见性分析**
   - 计算指定经纬度地面站的可见时段
   - 可自定义最小仰角（默认 10°）
   - 展示可见时段表格（开始/结束时间、持续时间、最大仰角等）

6. **iCalendar 导出**
   - 将卫星可见时段导出为 .ics 日历文件
   - 包含详细的过境信息

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行应用

```bash
python app.py
```

然后在浏览器中打开 `http://localhost:8080`

## 使用说明

1. **加载卫星数据**
   - 方法一：在 NORAD ID 输入框输入卫星编号（如 ISS: 25544），点击"获取 TLE"
   - 方法二：上传本地 TLE 格式文件

2. **查看轨道**
   - 加载卫星后，地图会显示未来 7 天的地面轨迹和当前位置
   - 下方 3D 视图展示地球和卫星轨道，可鼠标拖拽旋转、滚轮缩放

3. **计算可见性**
   - 输入地面站的纬度、经度和最小仰角
   - 点击"计算可见性"，查看未来 7 天的过境时段
   - 点击"导出 iCalendar"将结果导出为日历文件
