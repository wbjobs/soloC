# 梯形图编辑器使用指南

## 概述

PLC网关系统集成了一个功能强大的梯形图可视化编辑器，支持拖拽式编程、元件库管理、逻辑编译和PLC二进制文件下载。

## 功能特性

### 🎨 可视化拖拽编辑
- 直观的拖放界面，元件从侧边栏拖入梯级
- 支持无限添加梯级
- 可视化的连接线和电源轨
- 支持缩放操作

### 📦 元件库

#### 触点元件
- **常开触点 (NO)** - 条件为真时导通
- **常闭触点 (NC)** - 条件为假时导通

#### 线圈元件
- **输出线圈** - 标准输出
- **取反线圈** - 输出状态取反

#### 定时器/计数器
- **接通定时器 (TON)** - 输入接通后延迟输出
- **加计数器 (CTU)** - 输入脉冲计数

#### 逻辑元件
- **并联分支** - 实现OR逻辑

### 📋 模板系统

内置三个常用梯形图模板：

1. **自锁电路** - 经典的启动-停止-保持电路
2. **互锁电路** - 两个输出互锁，防止同时动作
3. **星三角启动** - 电机星三角降压启动控制电路

### 🔧 编译功能
- 梯形图逻辑语法检查
- 生成指令表 (IL)
- 检查输出有效性（梯级必须以输出结尾）

### 💾 二进制下载
- 生成PLC可执行二进制文件 (.plp)
- 包含校验和验证
- 标准格式：LDPL (Ladder Diagram Programming Language)

## 使用方法

### 1. 访问编辑器
从主面板右上角点击 **"⚡ 梯形图编辑器"** 按钮进入

### 2. 创建梯形图

#### 方式一：使用模板
1. 点击左侧侧边栏的模板
2. 系统自动加载预定义的梯形图
3. 根据需要修改地址和参数

#### 方式二：手动创建
1. 从左侧元件库拖拽元件到梯级
2. 双击元件编辑地址和属性
3. 点击 **"+ 添加梯级"** 增加新梯级
4. 点击元件上的 × 按钮删除元件

### 3. 元件编辑
双击任何元件打开编辑对话框：
- **地址/标签**: 如 I0.0, Q0.0, M0.0, T0 等
- **预设值**: 定时器和计数器的预设值（毫秒）

### 4. 编译和下载
1. 点击顶部 **"编译"** 按钮检查语法
2. 查看右下角状态面板的编译结果
3. 点击 **"下载二进制文件"** 生成 .plp 文件

## PLC二进制文件格式 (.plp)

### 文件头结构 (16字节)
```
偏移  长度  说明
0x00  4    魔数: 0x4C44504C ("LDPL")
0x04  2    版本: 0x0100 (v1.0)
0x06  2    保留
0x08  4    指令总数
0x0C  4    保留
```

### 指令结构 (16字节/指令)
```
偏移  长度  说明
0x00  1    操作码 (Opcode)
0x01  8    地址 (ASCII)
0x09  4    预设值/参数
0x0D  1    梯级编号
0x0E  1    保留
```

### 操作码定义
| 操作码 | 类型 | 说明 |
|--------|------|------|
| 0x01 | 常开触点 | Load Normally Open |
| 0x02 | 常闭触点 | Load Normally Closed |
| 0x03 | 输出线圈 | Output Coil |
| 0x04 | 取反线圈 | Negated Output |
| 0x05 | 接通定时器 | Timer On |
| 0x06 | 加计数器 | Count Up |
| 0x07 | 并联分支 | Parallel Branch |
| 0xFF | 梯级结束 | End of Rung |

## 地址规范

### 输入寄存器 (I)
```
I0.0 - I0.7  输入字节0
I1.0 - I1.7  输入字节1
...
```

### 输出线圈 (Q)
```
Q0.0 - Q0.7  输出字节0
Q1.0 - Q1.7  输出字节1
...
```

### 中间寄存器 (M)
```
M0.0 - M0.7  位寄存器
MW0, MW2...  字寄存器
```

### 定时器 (T)
```
T0, T1, T2...  定时器编号
```

### 计数器 (C)
```
C0, C1, C2...  计数器编号
```

## 编程示例

### 自锁电路
```
梯级0:
    [I0.0] ──┤├── [I0.1] ──┤/├── [Q0.0] ──( )──
                    │
              [Q0.0] ──┤├──  (自锁触点)
```
- I0.0: 启动按钮
- I0.1: 停止按钮
- Q0.0: 输出线圈

### 互锁电路
```
梯级0:
    [I0.0] ──┤├── [Q0.1] ──┤/├── [Q0.0] ──( )──

梯级1:
    [I0.1] ──┤├── [Q0.0] ──┤/├── [Q0.1] ──( )──
```

## 最佳实践

1. **命名规范**: 使用有意义的地址命名，如 Start, Stop, Motor 等
2. **注释**: 在复杂逻辑处添加注释说明
3. **梯级结构**: 每个梯级完成一个逻辑功能
4. **输出位置**: 输出线圈始终放在梯级最右侧
5. **编译验证**: 下载前始终先编译验证
6. **保存备份**: 定期导出和保存梯形图逻辑

## 故障排除

### 编译警告
- **"梯级未以线圈结尾"**: 确保每个梯级最右侧是输出元件
- **"空梯级"**: 删除空梯级或添加元件

### 下载失败
- 检查网络连接
- 确保后端服务运行正常
- 查看浏览器控制台错误信息

### 拖拽不工作
- 确保从侧边栏元件库拖拽
- 检查浏览器是否支持HTML5拖拽API
- 刷新页面重试

## 扩展开发

### 添加新元件类型
1. 在 `elementTypes` 添加新元件配置
2. 定义新的操作码
3. 更新编译和二进制生成逻辑
4. 添加到元件库UI

### 自定义文件格式
1. 修改 `generatePLCBinary()` 函数
2. 更新文件头结构
3. 添加新的指令编码规则

### 集成仿真功能
1. 添加执行引擎解释指令
2. 实现I/O模拟器
3. 添加在线调试功能
